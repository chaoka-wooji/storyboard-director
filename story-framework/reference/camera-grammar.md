# 镜头语法参考手册

> 本参考文件为模块三「微观蒙太奇分镜」提供镜头选择的理论底座。在执行模块三时，应根据场景情绪和叙事意图，从以下分类中选择合适的景别、角度、运镜和焦点。

---

## 一、景别（Shot Size）—— 决定观众与主体的情感距离

| 景别 | 缩写 | 取景范围 | 叙事功能与情绪效果 |
|------|------|---------|------------------|
| 极远景 | ELS / EWS | 主体极小，环境占主导 | 孤立感、史诗感、让环境成为角色；常用于开场定场或结局留白 |
| 远景 | LS / WS | 全身 + 大量环境空间 | 建立空间关系和人物在环境中的位置；传达渺小、自由或迷失 |
| 全景 | FS | 全身，头脚紧凑入画 | 展示完整肢体动作和服饰；适合登场亮相和动作展示 |
| 中远景 | MLS / MWS | 膝盖以上 | 兼顾肢体动作与面部表情；站立对话和群像场面常用 |
| 牛仔镜头 | CS | 大腿中部以上 | 展示肢体语言 + 面部微表情；隐含力量感和对抗姿态 |
| 中景 | MS | 腰部以上 | 最通用的叙事镜头——对话、动作、互动平衡；不偏不倚的"信息镜头" |
| 近景 | MCU | 胸部以上 | 开始强调面部，但保留部分肢体语境；情感过渡镜头 |
| 特写 | CU | 面部主导，填满画面 | 情绪揭示、关键反应、重大叙事信息；观众与角色的情感连接点 |
| 大特写 | ECU | 眼睛 / 嘴唇 / 手指 / 物体局部 | 极度强调——通常是叙事转折点或核心符号的揭示 |

### 景别选择原则

- **由远到近 = 从陌生到亲密**：开场用远景建立场景，高潮用特写放大情绪
- **景别对比产生节奏**：宽→紧→宽（呼吸感），紧→更紧（紧张升级）
- **特写是稀缺资源**：全片只用 2-3 次最关键的情绪特写，滥用会丧失冲击力

## 二、镜头角度（Camera Angle）—— 决定观众与主体的权力关系

| 角度 | 情绪效果 | 典型使用场景 |
|------|---------|------------|
| 平视（Eye Level） | 中性、平等、客观、纪实感 | 日常对话、写实叙事、角色间平等互动 |
| 仰拍（Low Angle） | 强大、威严、英雄感、威胁感 | 英雄登场、反派亮相、权力反转 |
| 俯拍（High Angle） | 脆弱、弱小、被压制、孤立无援 | 角色受挫、陷阱暴露、上帝视角审判 |
| 鸟瞰（Bird's Eye / Overhead） | 渺小、宿命感、疏离感、全知视角 | 开场定场、命运转折、死亡象征 |
| 荷兰角（Dutch Angle / Tilt） | 不安、失衡、心理危机、世界倾斜 | 角色精神状态恶化、发现惊天秘密、梦境扭曲 |
| 过肩（Over-the-Shoulder） | 对话代入感、权力距离暗示 | 两人对话的标准覆盖镜头 |
| POV（Point of View） | 主观沉浸、完全认同角色视角 | 恐怖片"杀手视角"、关键发现/揭示 |
| 侧面（Profile） | 沉思、疏离、不愿暴露内心 | 角色独处反省、隐瞒真相、告别 |

### 角度选择原则

- **仰拍 = 仰视 = 力量给谁**；**俯拍 = 俯视 = 谁被压制**
- 同一场景中角度变化 = 权力关系变化：从平视→仰拍暗示角色获得了力量
- 荷兰角不要滥用：一个场景最多 1-2 次，否则观众会疲劳

## 三、运镜方式（Camera Movement）—— 决定时间与空间的感知节奏

| 运镜 | 叙事功能 | 速度-情绪对应 |
|------|---------|-------------|
| 固定（Static / Locked-off） | 客观观察、让表演说话、营造稳定感 | 静态 → 平静/孤独/凝重 |
| 推轨（Dolly In） | 情感逼近——拉近观众与角色的心理距离 | 缓慢 → 渐渐投入；快速 → 冲击/顿悟 |
| 推轨（Dolly Out） | 情感抽离——角色被环境吞没、告别、揭示更大场景 | 缓慢 → 疏离/失落；快速 → 震惊 |
| 横摇（Pan） | 信息揭示、跟随视线、水平空间展示 | 缓慢 → 悬念/期待；快速 → 警觉 |
| 纵摇（Tilt） | 垂直空间展示、展示高度/深度/落差 | 缓慢 → 敬畏；快速 → 坠落感 |
| 变焦（Zoom） | 注意力强制聚焦（比 Dolly 更生硬，带有"机械感"） | 缓进 → 关注；急推 → 震惊 |
| 推轨变焦（Dolly Zoom / Vertigo Effect） | 心理失衡、认知崩塌、世界扭曲 | 背景压缩/拉伸 → 不安/恐惧 |
| 跟拍（Tracking） | 跟随角色旅程、保持时空连续 | 速度与角色同步 → 并肩感 |
| 手持（Handheld） | 临场感、混乱、紧迫、纪实风格 | 微晃 → 真实感；剧烈晃动 → 恐慌/战斗 |
| 弧线（Arc Shot） | 环绕主体——强调其中心地位或孤立 | 全弧 → 英雄时刻/顿悟 |
| 甩镜（Whip Pan/Tilt） | 快速转场、能量爆发、视点切换 | 极快 → 急促/紧张/节奏突变 |

### 运镜选择原则

- **固定 = 冷静/客观**，**运动 = 投入/主观**
- Dolly（物理移动）和 Zoom（光学变化）的心理感受截然不同：Dolly 让观众"走入画面"，Zoom 让观众"被拉过去"
- 运镜速度与情绪强度成正比：越慢越凝重，越快越紧张

## 四、焦点控制（Camera Focus）—— 决定观众注意力的落点

| 技巧 | 叙事功能 | 典型使用场景 |
|------|---------|------------|
| 浅焦（Shallow Focus） | 隔离主体，世界模糊只剩TA一人 | 主观孤独、深情对视、关键独白 |
| 深焦（Deep Focus） | 前景和背景同时清晰——环境与人物同等重要 | 群像调度、多线并行叙事、社会写实 |
| 变焦转移（Rack Focus） | 在同一画面中强制切换注意力落点 | 角色发现关键信息、同一镜头内"A 看 B"到"B 看 A" |
| 柔焦（Soft Focus） | 梦幻、回忆、怀旧、朦胧美感 | 闪回/回忆片段、浪漫时刻、即将醒来/昏迷 |
| 移轴（Tilt-Shift） | 极端风格化、心理异常感、微观世界感 | 角色精神状态异常、将现实场景"玩具化" |

## 五、情绪→镜头映射速查表

执行模块三时，根据目标情绪快速选用镜头组合：

| 目标情绪 | 推荐景别 | 推荐角度 | 推荐运镜 | 推荐焦点 |
|---------|---------|---------|---------|---------|
| 孤寂/疏离 | LS → WS | 俯拍 / 鸟瞰 | 固定 / 缓慢 Dolly Out | 深焦（人小环境大） |
| 紧张/焦虑 | MCU → CU（反复切） | 荷兰角（微倾 5-8°） | 手持微晃 / 缓慢 Dolly In | 浅焦 + Rack Focus |
| 力量/自信 | CS → FS | 仰拍 | 固定 / 弧线 | 浅焦 |
| 亲密/温暖 | CU | 平视 | 缓慢 Dolly In | 浅焦 / 柔焦 |
| 恐惧/威胁 | MCU → ECU | 俯拍（受害者）/ 仰拍（威胁源） | 手持 / 急促跟拍 | 浅焦 + 前景遮挡 |
| 悲伤/失落 | MS → CU | 俯拍 | 固定 / 缓慢 Dolly Out | 浅焦 / 柔焦 |
| 兴奋/激动 | MS → FS（切宽） | 平视 / 仰拍 | 手持 / 甩镜 / 快速跟拍 | 深焦 |
| 沉思/反省 | CU（侧面） | 平视（侧面/Profile） | 固定 | 浅焦 |
| 悬疑/悬念 | WS → MCU（缓慢推近） | 平视 / 过肩 | 缓慢 Pan / 缓慢 Dolly In | 深焦 → Rack Focus |
| 转场/过渡 | WS | 鸟瞰 / 远景 | 缓慢 Pan / Dolly Out | 深焦 |
