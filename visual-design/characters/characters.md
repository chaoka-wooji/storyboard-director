# 人类角色卡模板

## 适用范围

所有人类角色（不限性别、年龄）。若角色在故事中有服装/外观变化，在对应属性行中分段列出变化时间点。

## 输出格式

```
### 角色卡：{角色名称}

| 属性 | 设定 |
|------|------|
| 姓名/年龄 | [如：小林，13岁] |
| 身高/体型 | [如：155cm，偏瘦，四肢修长，肩宽约35cm] |
| 发型/发色 | [如：黑色高马尾，发长肩胛骨，碎刘海齐眉，鬓角散碎发丝约3cm] |
| 脸型 | [如：鹅蛋脸，下颌线柔和但咬肌微凸，额头饱满] |
| 眉毛 | [如：自然眉形，眉峰平缓，眉尾不散，颜色深棕] |
| 眼睛 | [如：杏眼，内眼角尖外眼角圆，虹膜深棕近黑，轻微下三白] |
| 鼻子 | [如：鼻梁挺直，鼻头小巧，鼻翼窄] |
| 嘴唇 | [如：上唇薄下唇略厚，唇色浅粉，放松时微张约0.5cm] |
| 肤色 | [如：偏白但被日晒后呈小麦色，鼻梁和颧骨有细微晒斑] |
| 标志特征 | [如：左手腕有一颗小黑痣，左膝外侧有约2cm旧疤痕] |
| 服装（时间段1） | [如：白色网球连衣裙，圆领无袖，腰部收褶，裙摆A字形长约40cm。材质速干涤纶，微透光可见内衬] |
| 服装（时间段2） | [如有变化如：同上，但胸口和腋下汗渍大面积扩散呈半透明] |
| 服装（时间段3） | [如：同款淡蓝色网球连衣裙，干净无汗渍] |
| 鞋/袜 | [如：蓝色Asics网球鞋（蓝白配色），白色短筒运动袜（袜口距脚踝约5cm）] |
| 配件 | [如：左手腕黑色运动手环，右手无饰品。可选：眼镜/帽子/围巾/手套等] |
| 核心配色 | [如：白色+蓝色+小麦肤色，整体明快清爽] |
| 出场镜头 | [如：镜1-17，镜21-24，镜28-30，镜34-54] |
```

## 填写指南

- **脸型/眉毛/眼睛/鼻子/嘴唇**：这是图生视频中角色一致性的关键锚点，必须精确描述。这些面部特征在后续模块五文生图和模块六图生视频中被用作跨镜头角色锁定依据
- **服装按时间段分行**：若故事跨越多个时间（如第一天/第二天），每个时间段独立一行
- **标志特征**：能帮助 AI 锁定角色身份的特殊标记（痣、疤痕、胎记、纹身等）
- **核心配色**：概括角色的主色调，便于控制跨镜头色彩一致性
