# 动物角色卡模板

## 适用范围

故事中出现的非人类角色（宠物、野生动物、幻想生物等）。

## 输出格式

```
### 角色卡（动物）：{角色名称/物种}

| 属性 | 设定 |
|------|------|
| 物种/品种 | [如：金毛寻回犬，成年雄性] |
| 体型/体重 | [如：肩高60cm，体长约100cm，体重约30kg] |
| 毛色/皮毛 | [如：全身金黄色长毛，背部颜色较深趋近棕金色，腹部和四肢内侧为浅奶油色。毛发质感浓密蓬松，尾部毛发尤其丰厚呈扇形散开] |
| 头部特征 | [如：头骨宽阔，吻部长度中等，鼻镜黑色湿润，耳朵自然下垂长度约15cm] |
| 眼睛 | [如：深棕色圆眼，眼白少，眼神温驯。内眼角下方有约1cm深色泪痕线] |
| 尾巴 | [如：尾根粗尾尖细，自然状态下呈微弯上扬约30°，兴奋时高举并大幅度左右摇摆] |
| 标志特征/伤痕 | [如：右耳边缘有一处约1.5cm的V形缺口，左前腿肘部有约3cm浅色旧疤痕] |
| 项圈/配件 | [如：红色尼龙项圈（宽约2.5cm），银色金属名牌吊坠（圆形直径约2.5cm）] |
| 核心配色 | [如：金棕色+奶油白+红色项圈] |
| 出场镜头 | [如：镜8，镜12-15，镜20-22] |
```

## 填写指南

- 动物的外形属性维度与人类不同，重点描述：毛色/皮毛纹理、体型比例、标志性姿态
- 如有项圈、衣物、缰绳等配件，需单独列出
- 若动物在故事中有外观变化（如受伤、沾泥、剃毛），分行列出时间点
