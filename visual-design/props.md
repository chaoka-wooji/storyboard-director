# 道具卡模板

## 适用范围

在画面中明确出现且有视觉表现的物体（运动器材、工具、食物、饰品、电子设备等）。按道具逐一输出。

## 输出格式

```
### 道具卡：{道具名称}

| 属性 | 设定 |
|------|------|
| 名称 | [如：Wilson网球拍] |
| 材质/颜色 | [如：碳纤维拍框（黑白涂装），聚酯硬线（黄色），拍柄黑色胶皮] |
| 尺寸 | [如：标准成人拍长68.5cm，拍面100平方英寸] |
| 细节 | [如：拍喉处有蓝色Wilson logo，拍柄底盖有红色W标志。拍框顶部有几道轻微擦痕] |
| 出场镜头 | [如：镜3，镜5，镜7-10，镜13...] |
```

## 填写指南

- **材质/颜色**：精确到材质类型（碳纤维/不锈钢/塑料/橡胶等）和颜色涂装
- **尺寸**：尽量给出准确数字（基于实物常识或合理估算）
- **细节**：logo位置、磨损痕迹、特殊标记——这些决定了道具在跨镜头中的辨识度
- 去重后逐一输出所有道具卡，不得遗漏
