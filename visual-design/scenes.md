# 场景卡模板

## 适用范围

故事中出现的所有场景（室内/室外、自然/人工环境）。按场景逐一输出。

## 输出格式

```
### 场景卡：{场景名称}

| 属性 | 设定 |
|------|------|
| 地点名称 | [如：社区网球场（室外硬地）] |
| 空间描述 | [如：标准单打场地（23.77m×8.23m），深蓝色硬地地面，白色界线宽5cm。场地围网高3.5m，墨绿色钢网菱形网格约4cm。场边有长条木制长椅，灰色帆布遮阳棚。练习墙在场地东侧，绿色涂料墙面高3m宽6m] |
| 时间/光线 | [如：清晨→正午→夕阳→夜间灯光。详见各镜头] |
| 主色调 | [如：深蓝（地面）+ 墨绿（围网）+ 暖橙（夕阳/灯光）] |
| 关键环境元素 | [如：围网外一排梧桐树，发球机（黄色Lobster品牌），4根8m高灯柱（金属灰）] |
| 氛围关键词 | [如：炎热/空旷/孤独/坚韧] |
| 出场场面 | [如：场面1-18] |
```

## 填写指南

- **空间描述**：精确到尺寸（米/厘米），包含材质和颜色
- **时间/光线**：标注场景跨越的时间段和对应的光线条件（自然光/灯光方向、色温）
- **关键环境元素**：列出场景中固定存在的大型物体，这些将在所有该场景的镜头中反复出现
- **氛围关键词**：用于后续文生图 Prompt 的风格定位
